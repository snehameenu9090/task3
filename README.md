# Python Basic Programs Bundle - Task 3

Welcome to **Task 3** of the Python programming repository. This project is a comprehensive collection of fundamental Python scripts designed to solve real-world mathematical, financial, and health-related logic problems using clean, reusable, and efficient code practices.

This repository serves as a practical demonstration of core programming concepts such as **Functions with Parameters, Conditional Statements (`if-elif-else`), Loops (`for` and `while`), User-Input Handling, and Advanced Recursion**.

---

## 🚀 Projects & Features Included

Every script in this task is self-contained, prompts the user for real-time input, and provides instant formatted outputs.

### 1. ⚖️ Body Mass Index (BMI) Calculator (`bmi.py`)
- **Description:** A reusable health utility function that calculates a user's BMI based on weight (kg) and height (meters).
- **Logic:** Uses the standard formula: $$BMI = \frac{\text{weight}}{\text{height}^2}$$
- **Features:** Categorizes the calculated value into standard health zones: *Underweight, Normal Weight, Overweight, or Obese*.

### 2. 💰 Equated Monthly Installment (EMI) Calculator (`emi.py`)
- **Description:** A financial utility script created to determine the monthly repayment amount for loans.
- **Features:** Accepts the Principal loan amount, annual Interest Rate, and Tenure (in months/years) to compute the exact EMI breakdown instantly.

### 3. 🔢 Fibonacci Series Generator (`fibonacci.py`)
- **Description:** A mathematical script focused on generating the famous Fibonacci sequence.
- **Features:** Asks the user for the number of terms and uses optimized looping structures to print the progressive series sequence cleanly.

### 4. 🔢 Prime Number Identifier (`prime.py`)
- **Description:** A logical checker built to test whether a given integer is a Prime or Composite number.
- **Features:** Implements a highly efficient division-check loop to process user-inputted numbers and return instant mathematical validation.

### 5. 🔄 Recursive Factorial Engine (`factorial_recursion.py`)
- **Description:** A program that calculates the factorial of a non-negative integer.
- **Logic:** Demonstrates the power of **Recursion** by making a function call itself ($n! = n \times (n-1)!$) until it reaches the base case ($0! = 1$)
